Комплект программ для работы с акциями и фьючерсами Московской биржи.
1. Программа для скачивания данных открытого интереса 
фьючерсов Московской биржи  со страницы ежедневных данных. 
Доступ доступен только пользователям личного кабинета.
В папке должен находиться файл <config.py>.
Где должны быть переменные: 
<username=*******> 
<password=*******>

2. Программа для заполнения базы данных SQLite котировками из папки с файлами csv.
   Названия файлов должны соответствовать названию таблиц.



Файл oi_histor.ру предназначен для скачивания доступной
истории одного инструмента.
